// useGraphStore — add to State
palette: 'mint' | 'iris';
setPalette: (p: 'mint' | 'iris') => void;

// in create()
palette: (localStorage.getItem('godview.palette.v1') as 'mint' | 'iris') ?? 'mint',
setPalette: (p) => {
  localStorage.setItem('godview.palette.v1', p);
  document.documentElement.style.setProperty('--mint',
    p === 'mint' ? '#5EEAB7' : '#B5A0FF');
  set({ palette: p });
},

Reference: see Atlas.html head <style> block — every accent uses var(--mint) so a single CSS-variable swap retints the whole interface.

SURFACE .06 Palette toggle

The toggle is two swatches in the wordmark column. Mint is default, Iris is the alternate. Preference persists. Every accent-coloured pixel in the entire app changes the moment you click.

CSS variable
--mint is the single source of truth. Mint = #5EEAB7, Iris = #B5A0FF. The name stays "mint" even when iris is active — that's the variable; iris just rebinds it.
READY

All shader materials hold a uColor uniform. On palette change, walk every points / line material and update it in one effect.

Component
